Login page

You are now logged in to edit things on Christopher Hogwood’s web site. Things which are now editable will have a “click to edit” link beneath them. Click here to return to the home page to navigate round the site and edit things.

By default you will remain logged in (and able to edit things) until you close the browser. If you want to explicitly log out, so that you can see what the site looks like without the “click to edit” links, then click this link: log out.

Internet explorer — warning

You appear to be using Internet Explorer to look at this site.

There are some significant differences between Internet Explorer and most other browsers, though these are starting to reduce with the arrival of Internet Explorer version 9.

These are not normally obvious in looking at web sites, but make a big difference to how newer facilities are interacted with by the scripting on a web site. This applies to some of the things used to enabling of the editing of this site. Rather than spend lots of extra time that editing also works in Internet Explorer, it seems wiser to disable editing with Internet Explorer for the time being.

Please come back to this page with another browser, such as Firefox.

This restriction only applies to editing the site. Things will look as good if people visit the site with Internet Explorer as if they use another browser.

 

Instructions

The editing facility applies only to those parts of the web site which have been explicitly set to be editable. Going back to the main site from this page, you will see a band under each thing which can be made editable links saying “click to edit” and “save”, and for things following a preset pattern, there are also links saying “new” and “delete”.

Clicking “click to edit” makes the entry editable — it is possible to type in new words, or to highlight some words and then use one of the links below to alter it.

Changing text

As a general rule of thumb, it is best to add new text and then delete what is there.

The reason for this is that if you delete all of the text for something, you are likely also to delete the marker that says where the new text should go. I have done some things to mitigate this, but is hard to more without making things look very different when they are being edited.

Moving things up and down

The pink boxes you click to edit things are at the bottom of each editable chunk of the page. At the left are arrows pointing up and down: clicking one of these will cause the section in question to swap places with the one above or below. This is a way, for example, of moving discography entries or news stories.

Images

The buttons at the bottom of an area being edited make changes to the text currently highlighted. To add an image, highlight some text click one one of the buttons to add images, and the text highlighted will be replaced with the image (aligned left, right or centred depending on which button you clicked).

As the photo replaces the text that was highlighted, people often type a few random characters to give themselves something to replace with the photo.

The photo will appear at the same size aas it is on your computer. To change the size of the photo, you should re-size it before uploading. Lots of graphics software can do this — the program people often talk about is Photoshop, which is qutei expensive, but there is a free, open-source alternative called Gimp available from www.gimp.org. A potential future upgrade to this editing facility would be to make the web site do the re-sizing, but often people get better results if they crop and re-size a photo in graphics software.

To delete an image all you need to do is click on the image and press delete (to remove it) or start typing (to replace it with words). To replace one image with another highlight the image and click one of the buttons to add an image.

Borders: Sometimes there is a case for putting a dark line round the edge of an image — typically if its background is so pale that it is not easy to distinguish from the background colour. If you click the checkbox for a dark border, then a dark border is added (if you uncheck the tick box the border goes away). You may need to click to end editing and then click to edit again before this becomes available.

Captions: the word “CAPTION” appears below all newly-uploaded images — which can be edited to the words actually wanted for the caption.

Secondary images

Sometimes it is good to be able to click on an image and see a large version. This is enabled by clicking on an image and the clicking the “Add secondary image” button. Thought the large and small versions are of the same thing, they need to have different names (otherwise the large one will over-write the small one). One way to do this is to add a suffix like “-big” to the file for the large version.

Paragraphs

To turn some text into a new paragraph, highlight it and click “paragraph”; clicking “quote paragraph” creats a new paragraph with the text right and left indented.

Special characters

It isn’t always as easy as it might be to get opening or closing single or double quotes, endashes or emdashes when typing into a web browser. To make these available some “special characters” are provided. To use these, highlight something in the text and click on one of the special characters, which will then replace what was highlighted.

“Show maincontent” and “Show content”

These are provided for debugging purposes and it will make sense to remove them befor ethe site goes live.

Saving or discarding edits

The edits you have made will only be saved when you click “save”. If you have been experimenting and want to discard what you have been doing, it is enough simply to reload the page.

Archive

Stories on the news page have a “to archive” option when available for editing: clicking on this removes the story from the news page (and menu) and provides menus for adding it to the archive.
